    \section{The Raspberry Pi}
    The Raspberry Pi is a low-cost general purpose computer made to allow easy access into computing for students and children, as well as being a great tool for projects that require a more complex computing platform than for example an MCU like the Atmel AVR yet still being easy to use unlike an FPGA. ~\cite{RaspberryPi}
